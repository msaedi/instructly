## Visual Specifications

### Line Style
- **Stroke width**: 1.5px
- **Color (unselected)**: #6B7280 (grey)
- **Color (selected)**: #000000 (black)
- **No fill**: All icons are outlines only
- **Corner radius**: 2px where applicable

## Interaction States

1. **Default**: Grey outline (#6B7280)
2. **Hover**: Darker grey (#374151)
3. **Selected**: Black (#000000) + yellow bar below
4. **Active/Pressed**: Slight scale (0.95)

## Benefits of Minimal Design

- **Modern & Clean**: Matches contemporary UI trends
- **Scalable**: Works at any size
- **Accessible**: High contrast, clear shapes
- **Fast Loading**: SVG icons are tiny file sizes
- **Consistent**: All icons have same visual weight
- **Professional**: Elevates the platform's perceived quality

These minimal icons will create a sophisticated, professional look similar to platforms like Airbnb, Uber, and other modern services.
