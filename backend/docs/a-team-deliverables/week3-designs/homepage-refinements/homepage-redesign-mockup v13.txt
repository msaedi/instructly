## Implementation Notes:

1. **Icons**:
   - Use minimal black & white SVG line-art icons
   - 24x24px, 1.5px stroke weight
   - No fill, outline only
   - See "Minimal Black & White Category Icons" artifact for SVG code

2. **Yellow Search Button**:
   - Exactly like TaskRabbit's green button but yellow
   - Circular end cap with search icon
   - Slight shadow for depth

3. **Category Bar**:
   - Continuous underline across all categories
   - Selected category gets yellow highlight bar
   - Icons simplified and greyed out
   - Equal spacing between categories

4. **Typography**:
   - "Instant learning with" - lighter weight
   - "iNSTAiNSTRU" - bold, slightly larger
   - Clear visual hierarchy

5. **Spacing**:
   - More breathing room above headline
   - Better proportions throughout
   - Follows TaskRabbit's generous spacing
