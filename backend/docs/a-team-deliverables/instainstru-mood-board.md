# InstaInstru Visual Mood Board

## Color Palette
- **Primary Blue**: #0066CC (Trust, education, NYC)
- **Accent Yellow**: #FFD700 (NYC taxi, energy, learning)
- **Success Green**: #00A86B (Available, confirmed)
- **Urgent Red**: #DC143C (Last spots, deadlines)
- **Neutral Grays**: #333333, #666666, #F5F5F5

## Typography
- **Headers**: Helvetica Neue Bold (like MTA signs)
- **Body**: Inter or SF Pro (clean, readable)
- **Numbers**: Tabular figures for prices/times

## Visual Elements

### NYC-Inspired
- Subway tile patterns for backgrounds
- Grid layouts (like NYC streets)
- Borough-based color coding

### Mobile Gestures
- Swipe between days
- Pull down to refresh availability
- Long press to preview instructor
- Pinch to zoom calendar

### Micro-animations
- Button press: Quick scale down/up
- Loading: Pulsing subway dots ...
- Success: Check mark draw-in
- Calendar: Smooth slide transitions

## Photography Style
- Natural light, NYC locations
- Diverse, real students and instructors
- Action shots (teaching moments)
- Authentic, not stock photos
