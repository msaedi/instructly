## Interaction Patterns

### Tab Navigation
- Click tab to switch content
- Active tab highlighted with accent color
- Smooth transition between tabs
- URL updates to allow deep linking (e.g., /account/billing)

### Favorite Instructors
- Auto-populated from booking history
- Click instructor card to view profile
- Remove button on hover/tap
- Maximum of 10 favorites

### Form Behaviors
- Auto-save for preferences
- Explicit save for critical changes
- Validation on submit
- Clear success/error messages

### Card Actions
- Hover states for interactive elements
- Click to expand transaction details
- Swipe to delete on mobile (with confirmation)

## Visual Specifications

### Colors
- Active tab: #FFD700 (brand yellow)
- Inactive tab: #666
- Success: #4CAF50
- Error: #F44336
- Borders: #E0E0E0

### Typography
- Tab labels: 16px, medium
- Section headers: 18px, bold
- Body text: 14px, regular
- Mobile: -2px from desktop

### Spacing
- Tab padding: 16px vertical, 24px horizontal
- Section spacing: 32px between major sections
- Card padding: 20px
- Mobile: 16px standard padding

- **Lesson Milestones**: 5, 10, 25, 50, 100 lessons
- **Streak Achievements**: 3, 7, 30 day streaks
- **Subject Explorer**: Try 5 different subjects
- **Top Student**: 5-star average rating
- **Early Bird**: Book morning lessons
- **Weekend Warrior**: Weekend lesson streak

## Handoff Notes

1. **Stats Bar**: Dismissible, shows progress to next achievement
2. **Multiple Addresses**: Allow custom labels, set default
3. **Credit Display**: Always show expiration date prominently
4. **Payment Methods**: Clear default indicator, easy switching
5. **Favorite Instructors**: Automatically populate from booking history, allow manual removal
6. **Learning Preferences**: Simple focus on subjects, instructors, and time preferences
7. **Responsive**: Mobile-first with desktop enhancements
8. **Validation**: Real-time for critical fields (email, phone)
9. **Empty States**: Helpful messages when no data exists
